PSCompletions

一个补全管理模块，它能让你在 PowerShell 中更简单、更方便地使用命令补全。

关于模块的命令
关于补全的 json 文件结构
关于 PR (Pull Request)

多语言

支持多种语言，你可以随意在这些语言中切换

动态排序

补全会根据命令输入的历史纪录动态排序，让你能够更快、更舒适的获取命令补全

灵活的补全菜单

模块提供了一个补全菜单，它比 PowerShell 自带的补全菜单更方便、更好用

补全基于 json

模块通过 json 文件去定义补全的数据
